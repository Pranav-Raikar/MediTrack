## Key features

**Patient app**
- Email/password login and signup
- Add medicines with name, dosage, frequency, and reminder times
- Daily local push notifications вЂ” fires even when app is closed (IST timezone)
- Toggle switch to pause/resume individual reminders
- Log health vitals: blood pressure, blood sugar, weight, heart rate
- 7-day trend chart for each vital type
- View all diagnoses added by any doctor
- AI health summary, vitals analysis, drug interaction check (premium)
- Share a 6-digit code to link a doctor

**Doctor portal (web)**
- Separate doctor login with role-based access
- Sidebar list of all linked patients
- View patient's medicines, vitals, and full diagnosis history
- One-click AI clinical summary (Claude API) before a consultation
- Add diagnoses that appear on the patient's app instantly
- Firestore security rules ensure doctors only see patients who shared their code

**Monetisation**
- AdMob banner shows at bottom of all screens for free users
- AdMob interstitial shows max once per day (on vitals tab open)
- Razorpay checkout for в‚№99/month patient plan
- Razorpay checkout for в‚№2,999/month clinic plan
- On payment success в†’ Firestore marks `isPremium: true` в†’ features unlock instantly

## Setup (quick start)

```bash
# 1. Install packages
flutter pub get

# 2. Connect to Firebase (generates firebase_options.dart)
dart pub global activate flutterfire_cli
flutterfire configure

# 3. Set Claude API key (server-side only вЂ” never in app code)
firebase functions:secrets:set CLAUDE_API_KEY

# 4. Deploy everything
chmod +x deploy.sh && ./deploy.sh
```

See `LAUNCH_GUIDE.md` for full step-by-step instructions including
AdMob, Razorpay, Play Store submission, and clinic sales strategy.
